Specifications
The navigation options for the McGraw-Hill Higher Education Instructor's Resource CD-ROM are listed in the menu column to the left. Clicking on any of the options listed in the menu column will either display the contents of the respective folder in the main window, or will display additional information about that asset. For instance, clicking on Instructor's Manual will display the contents of the Instructor's Manual folder on this CD-ROM, while clicking on Computerized Test Bank will display an informational page about the computerized testing software.

To open an asset displayed in the contents window, select a file and click on it. The file will then open in the default program for that file type. Some common file types include Microsoft Word document (.doc), Microsoft PowerPoint presentation (.ppt), Adobe Portable Document Format (.pdf), and common image files such as JPEG and GIF.

At the bottom of each contents page is a link to the supplement folder itself. Clicking on that link allows you to browse the contents of that supplement folder contained on the CD-ROM.

Certain types of common files (PDF, for example) may require third-party programs in order to view those files properly. You can download each of those programs by visiting the Websites listed below. We regret that viewers may not be available for all of the file types across the various computer platforms.
