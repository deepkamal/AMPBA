* Installation Instructions *

If the CD-ROM does not start automatically, follow the instructions
below: 

1. Click on the Start button and select Run.
2. Type D:\Start_Here.exe and click OK (where D is your CD-ROM drive.)



* Specifications *

The navigation options for the McGraw-Hill Higher Education 
Instructor's Resource CD-ROM are listed in the menu column to the 
left. Clicking on any of the options listed in the menu column will
either display the contents of the respective folder in the main 
window, or will display additional information about that asset. 
For instance, clicking on Instructor's Manual will display the 
contents of the Instructor's Manual folder on this CD-ROM, while 
clicking on Computerized Test Bank will display an informational 
page about the computerized testing software.

To open an asset displayed in the contents window, select a file 
and click on it. The file will then open in the default 
program for that file type. Some common file types include 
Microsoft Word document (.doc), Microsoft PowerPoint presentation 
(.ppt), Adobe Portable Document Format (.pdf), and common image files 
such as JPEG and GIF.
